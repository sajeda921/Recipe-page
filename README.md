# Recipe Page

A static recipe page built with HTML & CSS as part of a Frontend Mentor challenge.  
Displays ingredients and preparation steps.  
Desktop layout only. 

## Project Structure
- index.html  
- style.css  
- images/  
- fonts/  

## How to View
1. Download or clone the repository.  
2. Open `index.html` in your browser.
